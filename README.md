# VaigaiGrid-Sustainable-Urban-Water-Vision
  Urban Flooding 

**PROBLEM STATEMENT**


Madurai- several low-lying areas face frequent flooding -poor drainage

• Existing methods-removing water after flooding

• There is a need for an intelligent and cost-effective system that can:

   – Predict flood risk using Machine Learning
   
   – Monitor drainage and water levels in real time
   
   – Automatically divert excess floodwater into recharge tanks
   
   – Use relay-based automation for quick response

**Scalability**
